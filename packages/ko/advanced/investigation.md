# 조사 페이즈를 도입하고 싶다

투표 페이즈와 단서 배포를 결합하여 조사 페이즈를 구현할 수 있습니다.

덱을 사용한 조사에 대해서는、[이 페이지](../basic-features/decks.md)를 참조해 주세요。

## ① 페이즈의 생성

페이즈 목록에서「페이즈 추가」를 클릭하고、「투표」 페이즈를 선택합니다。

생성한 투표 페이즈 안에, 조사 지점의 선택지를 설정합니다。

여러 플레이어가 같은 장소를 조사하면 얻을 수 있는 단서가 줄어들기 때문에, 각자가 어디에 투표했는지 열람할 수 있도록 설정해 두는 것이 좋습니다。

\\

또한、「선택 조건」의「이 선택지의 중복 투표를 금지」를 클릭하면, 동일한 선택지에 2명 이상이 투표하지 못하도록 할 수 있습니다。 한 사람당 하나씩, 겹치지 않게 조사하게 하고 싶은 경우에 유효합니다。

플레이 중 화면에서는, 누군가가 선택한 선택지는 다른 사람의 화면에서 옅게 표시됩니다。

※현재、「이 선택지의 중복 투표를 금지」를 다른 조건과「그리고」나「또는」로 연결할 수 없습니다。

## ② 단서의 생성

단서 목록에서「신규 추가」를 클릭하고, ①에서 설정한 조사 지점에 대응하는 단서를 생성합니다。

단서의 배포 설정을 전원분, 아래와 같이 구성합니다。

| 항목         | 내용                     | 상세                                                               |
| ---------- | ---------------------- | ---------------------------------------------------------------- |
| 조건1（필요한 경우） | 페이즈 진행・○○가 시작했다         | <p>단서를 배포하는 페이즈입니다。<br>연속되는 투표 페이즈 중 가장 마지막 페이즈나 토론 페이즈가 좋습니다。</p> |
| 조건2        | 투표 행동・○○가 투표 페이즈에서 △△에 투표했다 | ①에서 만든 투표 페이즈에서, 어떤 캐릭터가 해당하는 조사 지점에 투표했다는 조건을 설정합니다。              |
| 배포 방법     | 비공개 상태로 소유권을 넘긴다           | －                                                                |
| 배포 대상       | 캐릭터를 선택              | 조건2의 캐릭터와 일치시킵니다。                                               |

\\

전체 공개・공유・양도 등의 조건은「페이즈 진행・○○가 시작했다」에서 토론 페이즈를 지정해 두는 것이 좋습니다。
