# 기능 업데이트 목록

### 향후 기능 업데이트 예정

* 준비 중



### 2026년 3월

* 복합 페이즈를 사용할 수 있게 되었습니다

### 2026년 2월

* 화면 디자인에서 사용할 수 있는 아이콘이 추가되었습니다

### 2026년 1월

* 연결되지 않은 페이즈가 남아 있어도 동작 확인을 할 수 있게 되었습니다
  * 이에 따라 첫 번째 페이즈를 나타내는 「플레이 시작」 아이콘이 페이즈 진행 화면에 추가되었습니다
* 「텍스트」를 페이즈, 단서, 감상전 화면 등에 배치할 수 있게 되었습니다
* 액션 결과를 조건에 따라 다르게 표시할 수 있게 되었습니다

### 2025년 12월

* 컬러 테마와 배경 이미지가 통합되고, 폰트와 아이콘 변경 기능이 추가된 「[화면 디자인](../basic-features/effect/screen-design.md)」을 사용할 수 있게 되었습니다
* [우즈스튜디오 구현 의뢰 서버](https://discord.gg/v76QmZAXHA)가 개설되었습니다

### 2025년 10월

* [텍스트 그룹](../../../basic-features/textTab.md)을 사용할 수 있게 되었습니다
* 지문·대사의 자동 넘김과 읽는 사람의 중간 교대를 할 수 있게 되었습니다
* [탐색 페이즈](../basic-features/phase/basic/canvas.md)를 사용할 수 있게 되었습니다

### 2025년 9월

* 트리거와 액션에서 「플레이어의 마이크를 자동으로 음소거하기」를 사용할 수 있게 되었습니다
* 페이즈 시작이나 액션 실행 후 ●분/●초 뒤에 발동하는 지연 액션을 사용할 수 있게 되었습니다
  * 예: 토론 페이즈 시작 10분 후 자동으로 단서를 공개하기, 최종 토론 페이즈 종료 5분 전부터 밀담 룸 사용을 금지하기

### 2025년 8월

* 음량 조정 사양이 변경되어, 가능한 한 음질을 유지한 채 음량 조정이 실행되도록 개선되었습니다
* 트리거가 새로 추가되었습니다
  * 페이즈 강제 전환과 룸 강제 이동을 할 수 있게 되었습니다
  * SE·BGM에서 페이즈나 큐 이외의 재생 조건을 사용할 수 있게 되었습니다

### 2025년 7월

* 텍스트의 읽음 배지 사양이 업데이트되었습니다
* 덱에 여러 개의 토큰을 배치할 수 있게 되었습니다
* 리포트 화면에서 투표 결과를 볼 수 있게 되었습니다

### 2025년 6월

* 리포트 화면에서 페이즈 도달률을 볼 수 있게 되었습니다

### 2025년 5월

* 페이즈 진행 화면에서 자동 정렬을 사용할 수 있게 되었습니다

### 2025년 3월

* 페이즈 전환을 드래그 앤 드롭으로 변경할 수 있게 되었습니다
* 텍스트와 이미지 등도 표시 조건을 유지한 채 복제할 수 있게 되었습니다
* 목표에 표시 조건을 설정할 수 있게 되었습니다

### 2025년 2월

* 다음으로 넘어갈 수 있는 조건을 설정할 수 있게 되었습니다
* 기본 페이즈에 단서를 배치할 수 있게 되었습니다

### 2025년 1월

* 투표 결과를 어디에든 표시할 수 있게 되었습니다
* 배포 규칙에 배포 소재 URL을 넣을 수 있게 되었습니다
* 리딩의 대사·문장을 조건에 따라 건너뛸 수 있게 되었습니다

### 2024년 12월

* 페이즈 목록 화면에 필터 기능이 추가되었습니다
* [자유 서술 박스](../basic-features/text-fields.md)를 사용할 수 있게 되었습니다

### 2024년 10월

* 투표 블록의 앞뒤에 문장이나 이미지를 넣을 수 있게 되었습니다
* 액션 버튼으로 동영상을 재생할 수 있게 되었습니다
* 우즈스튜디오 URL이 변경되었습니다→https://editor.studio.uzu-app.com/

### 2024년 8월

* [컬렉션 기능](../top/collection.md)이 추가되었습니다
* 줄거리·등장인물의 비표시를 선택할 수 있게 되었습니다
* 텍스트 탭에서 「항상 읽음 상태로 표시」를 설정할 수 있게 되었습니다
* 커버 이미지에 [서브 이미지](../scenariohome/overview/sub.md)를 설정할 수 있게 되었습니다

### 2024년 6월

* 수수께끼 풀이용 템플릿과 1인용 템플릿이 추가되었습니다

### 2024년 5월

* 우즈스튜디오 내에서 음성·BGM 파일의 음량을 조정할 수 있게 되었습니다
* 접기/펼치기 블록을 사용할 수 있게 되었습니다
* [2회차+캐릭터](../basic-features/character/second-play-character.md)를 설정할 수 있게 되었습니다
* 덱 안의 단서를 [무작위 배포](../../../basic-features/decks.md#worandamunisuru)할 수 있게 되었습니다

### 2024년 4월

* 투표 페이즈에서 원하는 조건으로 복수 표를 투표할 수 있게 되었습니다
* 시나리오 도중에 캐릭터의 닉네임과 아이콘을 변경할 수 있게 되었습니다
* [유료 작품을 무료로 플레이할 수 있는 티켓](../scenariohome/marketing/#chikettokdono)을 발행할 수 있게 되었습니다

### 2024년 3월

* 토큰을 여러 종류 만들 수 있게 되었습니다
* 비밀번호 액션의 정답을 여러 개 설정할 수 있게 되었습니다
* 시나리오 태그를 우즈스튜디오에서 설정할 수 있게 되었습니다

### 2024년 2월

* 「누군가」 조건 설정을 할 수 있게 되었습니다
* 조건 설정에서 숫자 비교를 할 수 있게 되었습니다→자세한 내용은 [여기](../basic-features/condition.md)

### 2024년 1월

* 컬러 테마가 2종에서 9종으로 확장되었습니다(1/12)→자세한 내용은 [여기](../basic-features/effect/)
* 메인 룸의 제목을 조건 설정으로 변경할 수 있게 되었습니다(1/17)
* 배경 이미지를 설정할 수 있게 되었습니다(1/31)→자세한 내용은 [여기](../basic-features/effect/background.md)

### 2023년 12월

* 자유 응답 액션(원하는 문자를 입력하면 단서를 획득할 수 있는 기능)을 추가했습니다(12/7)
* 원하는 조건으로 문장의 표시·비표시를 전환할 수 있는 기능을 추가했습니다(12/15)
* 이미 배포된 단서를 회수할 수 있는 기능을 추가했습니다(12/19)

### 2023년 11월

* 그리고/또는 복합 조건을 설정할 수 있게 되었습니다(11/1)
  * 그리고/또는을 최대 3단계까지 사용할 수 있어 더 복잡한 조건 분기를 설정할 수 있습니다
* 토큰 조사 기능이 추가되었습니다(11/14)
  * 토큰을 사용해 덱에서 단서를 뽑는 방식의 조사를 구현할 수 있습니다
* 인원수별 템플릿을 공개했습니다(11/20)

### 2023년 10월

* 단서 배포 설정 업데이트(10/6)
* 단서 배포 조건에서 「누군가」를 선택할 수 있게 되었습니다(10/20)
  * 특정 선택지를 고른 임의의 캐릭터에게 단서를 배포하는 설정을 쉽게 만들 수 있게 되었습니다

### 2023년 9월

* 동작 확인·버전 생성 시 오류 표시가 명확해졌습니다(9/1)
  * 오류 위치로 클릭 한 번에 이동하는 기능이 구현되어 수정이 쉬워졌습니다
* 다크 모드를 설정할 수 있게 되었습니다(9/5)
  * 호러, 시리어스 계열 시나리오에 어울리는 테마 컬러를 사용할 수 있습니다
* 선택지 중복 투표를 금지하는 옵션이 추가되었습니다(9/7)
  * 조사 시 다른 사람과 같은 장소를 조사하지 못하도록 제한할 수 있습니다
* 미공개 단서의 존재를 소유자 이외의 사람에게 표시하는 옵션이 추가되었습니다(9/7)
  * 이른바 카드의 「뒷면」이 다른 사람에게 보이는 상태입니다
* 페이즈 진행 설정 화면에서 더블 클릭으로 해당 페이즈로 이동할 수 있게 되었습니다(9/7)
* 동작 확인에서 「다음 페이즈로 진행(전원분의 다음 버튼을 눌러 줌)」 버튼이 추가되었습니다(9/22)

### 2023년 8월

* 서버 증설로 속도가 개선되었습니다(8/2)
* 대규모 리디자인(8/3)
* 페이즈·텍스트·캐릭터·단서의 순서를 바꿀 수 있게 되었습니다(8/3)
* 구분선을 더 쉽게 사용할 수 있게 되었습니다(8/9)
* 투표 결과를 감상전 화면에 표시할지 선택할 수 있게 되었습니다(8/9)
* 공식 가이드를 릴리스했습니다(8/14)
* 스마트폰 편집을 지원하게 되었습니다(8/24)

### 2023년 7월

* 페이즈 전환 우선순위의 순서를 바꿀 수 있게 되었습니다(7/31)
* 기본 전환과 조건 전환을 쉽게 변경할 수 있게 되었습니다(7/31)

### 2023년 6월

* 우즈스튜디오 v2 릴리스(6/29)
