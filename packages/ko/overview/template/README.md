# 채워 넣기만 하면 마다미스 완성 템플릿

우즈 스튜디오에서는 템플릿을 활용해, 채워 넣기만으로 간단히 마다미스를 완성할 수 있습니다.

템플릿의 사용법에 대해 4인용을 예로 들어, [기본편](basic.md)과 [발전편](advanced.md)으로 나누어 설명합니다.
