# 내용만 채우면 머더 미스터리 완성 템플릿

우즈 스튜디오에서는 템플릿을 활용해, 내용을 채우는 것만으로 간단히 머더 미스터리를 완성할 수 있습니다.

템플릿 사용법은 4인용을 예로 들어 [기본편](basic.md)과 [응용편](advanced.md)으로 나누어 설명합니다.
