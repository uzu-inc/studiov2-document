# 키 비주얼 제작 규정

우즈 스튜디오에서 게시하는 시나리오의 키 비주얼(커버 이미지·타이틀 이미지)에 관한 규정을 제정했습니다. 

## ① 이미지는 가로세로 비율을 16:9로 제작할 것

16:9 이외의 비율로 제작된 키 비주얼은 앱 내에서 올바르게 표시되지 않습니다.

## ② 제목이 읽기 쉬운 대비를 갖도록 할 것

배경 이미지와 제목 문자 색의 대비가 약하면 가독성이 떨어지므로, 대비를 의식해 제작해 주세요. 

{% hint style="success" %}
『컨트라스트』란\
'대비'를 뜻합니다. 키 비주얼에서는 명도·채도·색상의 차이를 말합니다. 
{% endhint %}

<figure><img src="../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

### 개선 방법

▷ 글자색을 조정한다

▷ 제목 글자에 외곽선을 넣는다

▷ 제목 글자에 그림자를 추가한다

▷ 배경 이미지에 검정 또는 흰색을 옅게 겹쳐 씌운다

{% hint style="info" %}
**【발전적 주의점】**

위의 【나쁜 예】【좋은 예】에서는 「명도」「색상」을 조정해 읽기 쉽게 만드는 대응을 했지만, 주의점이 하나 있습니다. 

그것이 「**할레이션**」입니다. 

<img src="../.gitbook/assets/image (38).png" alt="" data-size="original">

할레이션이란 배색의 「명도」가 비슷하고, 동시에 「채도」가 높을 때 발생하는 눈이 따가워지는 상태를 말합니다. 시인성이 매우 나쁜 디자인이 되므로, 이 상태를 피하는 것이 바람직합니다. 
{% endhint %}

## ③ 크기를 축소해도 제목(주제)을 읽을 수 있을 것

<figure><img src="../.gitbook/assets/image (40).png" alt=""><figcaption></figcaption></figure>

### 개선 방법

▷ 폰트 두께를 굵게 한다

▷ 폰트 크기를 키운다(기준: 가로쓰기에서 10~15글자를 나열했을 때의 크기 이상)

{% hint style="success" %}
우즈 앱의 시나리오 목록 화면에서는 「160px ✕ 90px」으로 표시됩니다. 릴리스 관리 화면에 작게 표시되는 키 비주얼에서, 읽을 수 있는지 확인하면 이해하기 쉽습니다↓
{% endhint %}

<figure><img src="../.gitbook/assets/image (155).png" alt=""><figcaption></figcaption></figure>



## ④ 시나리오 상세에 표시되는 제목은 이미지 안에 전문을 모두 넣을 것

<figure><img src="../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

제목 이외의 문자를 넣는 경우에는, 제목 글자가 가장 눈에 띄는 폰트 크기가 되도록 해 주세요. 

## ⑤ 후리가나에 대하여

우즈 스튜디오에는 「후리가나 기입란」이 있지만, 이는 앱 내 히라가나 검색을 위한 것이며, 앱 내에서는 후리가나가 표시되지 않습니다. 

「후리가나를 제목에 넣는다」 「제목의 읽는 법을 작가 코멘트란에서 보충한다」의 어느 대응도 하지 않는 경우는, 키 비주얼에 후리가나를 포함해 주세요. 후리가나가 필요한 기준은 아래와 같습니다. 

{% hint style="info" %}
① 일반적으로 난독으로 여겨지는 읽기가 포함된 경우

② 여러 가지 읽는 방법이 있는 것

③ 고유명사

④ 당자

⑤ 일반적이지 않은 영어 단어·외국어
{% endhint %}

## ⑥ 사용 중인 소재가 저작권·퍼블리시티권을 침해하지 않을 것

{% hint style="info" %}
**저작권이란?**（간단한 설명）「저작물」を 창작한 자에게 부여되는, 자신이 창작한 저작물이 무단으로 복제되거나 인터넷상에서 사용되는 것을 금지할 수 있는 권리이다. 

키 비주얼에서는 권리자의 허락 없이 일러스트나 이미지 소재를 사용하는 것은 저작권 침해에 해당할 가능성이 높으니 주의하세요. 
{% endhint %}

{% hint style="info" %}
**퍼블리시티권이란?**（간단한 설명）

고객 흡인력을 갖춘 유명인의 성명·초상 등을 보호하는 권리이다. 

유명인이나 저명인의 사진을 무단으로 사용하는 것은 퍼블리시티권 침해에 해당하므로 주의하세요. 
{% endhint %}
