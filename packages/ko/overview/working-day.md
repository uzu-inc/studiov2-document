{% hint style="info" %}
위 영업시간 외에 심사 제출이나 폼 응답, 질문을 보내셔도 문제 없습니다. 다만 운영팀의 즉시 대응은 보장되지 않는 점 양해 부탁드립니다.
{% endhint %}

## 운영팀의 대응이 필요한 예

다음의 경우에는 운영팀의 대응이 필수이므로, 영업시간을 확인하신 후 여유를 두고 연락해 주세요.

**① v2로 이행할 때 등, 유료 공개 중인 작품의 무료 테스트 플레이 테이블 생성을 운영팀에 의뢰하는 경우**

다음 3가지를 영업시간 내에 운영팀에 공유해 주세요.

* 테스트 플레이를 위한 수정 및 버전 생성이 완료된 상태의 해당 시나리오 제목
* 테스트 플레이 시작 일시
* 관전자(작가)의 사용자 ID

{% hint style="info" %}
※연락을 잊었을 경우의 해결 방법

마케팅 화면에서 무료 티켓을 발행해 진행한다

일시적으로 시나리오 복사본을 만들어 테스트 플레이를 진행한다
{% endhint %}

**② v2로 이행할 때 등, 우즈에서 해당 시나리오를 이미 플레이한 사람들을 모아 테스트 플레이를 하고 싶은 경우(원래 플레이 완료한 사람은 테스트 플레이에 참여할 수 없습니다)**

다음 4가지를 영업시간 내에 운영팀에 공유해 주세요.

* 테스트 플레이를 위한 수정 및 버전 생성이 완료된 상태의 해당 시나리오 제목
* 테스트 플레이 시작 일시
* 관전자(작가)의 사용자 ID
* 참가자 전원의 사용자 ID

{% hint style="info" %}
※연락을 잊었을 경우의 해결 방법

일시적으로 시나리오 복사본을 만들어 테스트 플레이를 진행한다
{% endhint %}

**③ 주말·공휴일이나 장기 휴가에 신규 공개를 희망하는 시나리오의 공개 준비용 폼 제출**

예를 들어, 일요일 공개를 희망하는 시나리오의 폼을 토요일에 제출하신 경우에는 대응이 어렵습니다.

**④ 주말·공휴일 또는 장기 휴가 중에 신규 공개되는 시나리오에서, 예약된 공식 포스트의 내용이나 키 비주얼을 변경하고 싶은 경우**

변경이 있는 경우에는 직전 영업일의 영업시간 내에 연락해 주세요.
