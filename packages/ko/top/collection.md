# 컬렉션 기능

## 설정 방법

컬렉션 설정은 우즈스튜디오의 메인 화면(제작 시나리오 목록 화면)에서 진행합니다.

**① 왼쪽 메뉴에 있는「새 컬렉션」을 클릭합니다.**

**② 컬렉션 제목**（앱 내 검색에서 검색됨）**・설명문을 입력하고, 넣고 싶은 작품을 추가합니다.**\
&#xNAN;**※ 추가할 수 있는 것은 자신이 관리자인 작품만입니다.**

{% hint style="warning" %}
**주의 사항**

하나의 시나리오를 여러 컬렉션에 소속시킬 수는 없습니다.

「탐정 시리즈」「청춘 시리즈」와 같은 일반적인 단어를 컬렉션명으로 사용하고 싶을 때는 「○○의 탐정 시리즈」「청춘 시리즈(○○)」처럼 **다른 사람의 동명 시리즈와 구분될 수 있도록** 설정해 주세요.

이미 사용 중인 컬렉션명을 입력하면 오류가 표시되도록 할 예정입니다.
{% endhint %}

## 앱에서의 모습

앱에서는 시나리오 상세 화면의 키 비주얼 아래에 컬렉션 열람 버튼이 표시됩니다.

컬렉션명을 탭하면 컬렉션 상세를 확인할 수 있습니다. 컬렉션 상세에서는 「컬렉션에 들어 있는 작품의 **좋아요 수 합계**」 「컬렉션에 들어 있는 작품의 **작가 목록**」 「**작품 목록**」이 표시됩니다.

우즈스튜디오의 컬렉션 **편집 화면에서의 순서와 관계없이, 작품은 출시 순으로 정렬됩니다**.

## 관련 시나리오 기능과의 구분

{% hint style="info" %}
**컬렉션 기능의 사용 예**

・시리즈 작품을 묶기

・같은 작가의 같은 장르 작품을 묶기

・기획 작품을 묶기
{% endhint %}

{% hint style="info" %}
**관련 시나리오 기능의 사용 예**

・해당 시나리오를 플레이하기 위해 전제로 미리 플레이해야 하는 시나리오(예: 시리즈의 전작)

・해당 시나리오의 체험판 시나리오(예: 유료 시나리오의 무료 체험판)

・해당 시나리오의 다른 버전(예: GM 있음/없음)
{% endhint %}

**관련 시나리오 기능은 「내용적으로 관련이 없는, 단지 같은 작가의 시나리오」 「무관한 작품의 홍보」에는 이용하실 수 없습니다.**

작품 수가 많은 시리즈의 경우에는, 관련 시나리오에 전부를 올리는 대신 컬렉션 기능을 이용해 주세요. 바로 이전 1작을 관련 시나리오에 올리는 것은 문제 없습니다.
