# 페이즈 진행

페이즈 진행은 **페이즈 간의 순서**와 **분기**를 설정하는 화면입니다.

왼쪽 메뉴에서「페이즈」를 선택하고, 해당 화면의「진행 설정」을 클릭하면 편집 화면으로 이동할 수 있습니다。

\\

## 기본 조작

기본적인 조작은 「나열하기」와 「연결하기」 두 가지입니다。

페이즈와 페이즈는 아래와 같은 조작으로 서로 연결할 수 있으며, 어떤 페이즈 다음에 어떤 페이즈가 올지 설정할 수 있습니다. 선 위에 적힌 알파벳과 왼쪽 열에 표시되는 알파벳이 서로 대응합니다.

이미 연결되어 있는 선(전이)을 다른 곳으로 다시 연결하고 싶다면, 오른쪽 열에서 해당 알파벳을 찾아, 「・・・」에서 **삭제**를 선택하세요。

※ 스마트폰으로 열람하는 경우, 가로 모드로 보지 않으면 오른쪽 절반이 표시되지 않습니다。\\

## 전이의 종류

페이즈와 페이즈 사이의 선(전이)에는, 「**기본 전이**」와 「**조건 전이**」의 두 종류가 있습니다。

| 전이의 종류 | 모양        | 설명                                                                 |
| ----- | --------- | ------------------------------------------------------------------ |
| 기본 전이  | 검은 선에 검은 글자 | <p>「다음」을 탭/설정한 시간 경과로 전이됨<br>조건 전이 어느 것도 충족하지 않을 때 전이됨</p> |
| 조건 전이  | 파란 선에 파란 글자 | 조건을 만족하는 경우에만 전이한다                                                |

\\

하나의 페이즈에서 여러 개의 선(전이)을 설정하면, 반드시 하나는 「**기본 전이**」가 되고 나머지는 「**조건 전이**」가 됩니다. 투표에서 엔딩으로의 **분기**를 떠올리면 이해하기 쉬울 것입니다.

조건 분기가 여러 개일 때는 왼쪽 열에서 **위에서부터 순서대로 우선**됩니다. 알파벳은 단순한 식별 기호일 뿐이며, 우선순위와는 관련이 없습니다.

전이 조건을 설정하거나, 삭제할 때는 「・・・」를 누르세요。\\

분기 조건에 「그리고(かつ)」와 「또는(または)」의 복합(중첩)을 사용할 수도 있습니다. 자세한 내용은 [여기](../condition.md#gurpu)를 참조하세요.
