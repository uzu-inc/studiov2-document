# 기본 단계

기본 단계에서는**텍스트・이미지・덱・액션** 등의 표시가 가능합니다. 캐릭터 텍스트 읽어들이기, 토론, 조사, 설명 제시, 추가 정보 확인 등에 적합한 단계입니다. 



## 입력할 항목과 내용

| 항목                  | 내용                                                                                                     | 기입 예                         |
| ------------------- | ------------------------------------------------------------------------------------------------------ | --------------------------- |
| 관리 제목(선택)          | <p>작성자가 우즈 스튜디오에서 시나리오를 구현할 때 메모용으로 사용하는 제목입니다. <br>앱에는 반영되지 않습니다. </p>                                           | ○○ 단계의 다음・○○인 경우 등            |
| 제목(필수)            | 상단에 표시될 제목을 설정합니다.                                                                                    | 캐릭터 텍스트 읽어들이기・제1 토론 등       |
| 부제목(선택)          | <p>제목 아래에 검은 글자로 표시되는 문장을 설정합니다. <br>빈칸인 경우 표시되지 않습니다. </p>                                                      | 텍스트 탭을 열어주세요・토론을 시작해 주세요 등 |
| 캡션(선택)          | <p>제목 아래에 회색으로 표시되는 문장을 설정합니다. <br>빈칸인 경우 표시되지 않습니다. </p>                                                     | 토론 시간은 10분입니다・단서 공개가 가능합니다 등   |
| 소요 시간                | <p>단계 완료 조건을 '다음 버튼'으로 설정한 경우에는 대략적인 소요 시간, '제한 시간(타이머)'으로 설정한 경우에는 타이머 설정 시간이 됩니다.  </p><p>전자의 경우, 앱에는 표시되지 않습니다. </p> | 반각 숫자                        |
| 단계 완료 조건(필수)        | '다음' 버튼을 눌러 진행할지, '타이머'를 설정할지 결정합니다.                                                                       | －                           |

| 단계의 내용(필수)         | 공통 화면에 표시할 텍스트・이미지・덱・액션・투표를 설정합니다.                                                                     | 지시문・공통 정보・덱・액션・투표 등      |

## 단계의 내용에 대해

텍스트・이미지・덱・액션 등은 '+추가' 버튼에서 선택하여 추가합니다. 

<figure><img src="../../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

텍스트에서는 아래와 같은 문자 서식을 사용할 수 있습니다. <figure><img src="../../.gitbook/assets/image (116).png" alt=""><figcaption></figcaption></figure>



이미지에서는 이미지를 화면의 어디에 정렬할지(왼쪽 정렬·가운데 정렬·오른쪽 정렬), 어느 정도 크기로 표시할지 설정할 수 있습니다. 

<figure><img src="../../.gitbook/assets/image (117).png" alt=""><figcaption></figcaption></figure>



## 단계의 내용을 텍스트에도 배포하기

이 단계 중에 공통 화면에 표시한 정보를 텍스트 탭에도 배포하고 싶다(나중에 다시 확인할 수 있도록 하고 싶다)면 오른쪽 메뉴에서 설정합니다. 

단계의 내용과 동기화된 텍스트(텍스트 측에서 편집 불가·단계에서 편집하면 텍스트에도 자동 반영됨)를 추가하고 싶다면\
→ 단계 내용의 기록 「텍스트로 배포」를 ON으로 설정하기만 하면 됩니다. 

현재 입력된 단계의 내용을 복사한 텍스트(텍스트 측에서 편집 가능)를 추가하고 싶다면

1. 배포물 「＋추가」를 누른다
2. 텍스트를 선택한다
3. 배포 텍스트 항목을 연다
4. 「이 단계의 복사본」을 선택한다

동기화된 텍스트는 규칙 설명·공통 정보를 추가하고 싶을 때 등에, 복사한 텍스트는 개별적으로 편집을 가하고 싶은 내용일 때 편리합니다.
