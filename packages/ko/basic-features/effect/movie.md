# 무비

mp4 형식의 동영상을 업로드할 수 있습니다. ※10MB를 초과하는 파일은 업로드할 수 없습니다。



### 재생 조건

**무비 업로드 화면에서의 설정**\
페이즈, 큐(대사·지문)를 트리거로 설정할 수 있습니다. 무비의 스킵 버튼은 플레이어에 표시되지 않습니다。

**액션 설정 화면**\
액션을 실행하면 재생되도록 설정할 수 있습니다. 무비의 스킵 버튼을 플레이어에 표시할지 선택할 수 있습니다。
