# 배경 이미지

플레이 화면의 배경 이미지를 설정할 수 있습니다.

### 기본 배경

<figure><img src="../../.gitbook/assets/스크린샷 2024-01-31 14.53.39.png" alt=""><figcaption></figcaption></figure>

기본 배경은, 조건부 배경이 설정되어 있지 않거나 또는 조건부 배경의 조건을 만족하지 않을 때 계속 표시되는 이미지입니다. 이미지는 드롭하거나 클릭하여 설정할 수 있습니다.

기본 배경을 설정하지 않은 작품에서는, 컬러 테마에 맞춘 단색 배경이 자동으로 적용됩니다.



이미지를 설정하면 아래와 같이 됩니다. 동작 확인을 수행하면 배경 이미지가 설정되어 있음을 확인할 수 있습니다.

배경 이미지에는 컬러 테마에 맞춘 색상 블러(라이트 모드라면 흰색, 다크 모드라면 검은색 등)가 자동으로 적용됩니다. 블러는 어떤 배경 이미지를 업로드하더라도 텍스트를 읽을 수 있을 정도의 농도로 설정되어 있습니다. 머더 미스터리 앱으로서 텍스트를 읽을 수 있어야 한다는 전제가 있으므로, 블러를 OFF로 할 수는 없습니다.

### 조건부 배경

조건부 배경은, 임의의 조건을 만족했을 때 표시되는 배경입니다.

‘조건부 배경’ 부분에 이미지를 설정하면 조건을 설정할 수 있게 됩니다.

### 설정한 타이밍에 배경이 바뀌지 않나요? <a href="#why-not-change" id="why-not-change"></a>

배경 업로드 화면에서 왼쪽부터 순서대로, ‘배경 A: 페이즈 1이 시작됨’, ‘배경 B: 페이즈 2가 시작됨’이라는 배경을 설정했다고 가정합니다.

페이즈 2 시작 시점에 배경 A에서 배경 B로 바뀌길 바라지만, 이 설정으로는 바뀌지 않습니다.

페이즈 1 → 페이즈 2로 진행된 경우, ‘페이즈 1이 시작됨’이라는 조건은 페이즈 2가 시작돼도 여전히 충족된 상태이기 때문입니다. 조건은 더 왼쪽에 있는 이미지가 우선이므로, 배경 A의 조건이 충족되는 한 계속해서 배경 B로 바뀌지 않습니다.



개선 방법

* 배경 A의 조건을 「페이즈 1이 진행 중」으로 바꾸기
* 배경 A의 조건을 「페이즈 1이 시작되었고 페이즈 2에 진입하지 않았다」로 바꾸기



Tips

배경 이미지로 설정해도, 위쪽 부분은 캐릭터 표시로 가려져 플레이어에게 보이지 않습니다.
플레이어에게 보이는 부분의 비율을 고려하면 배경 이미지가 이상한 위치에서 잘리는 일이 없게 됩니다.
