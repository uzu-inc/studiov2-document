# BGM

BGM은 주로 특정 단계에서 반복 재생하는 음원에 사용됩니다.

<figure><img src="../../.gitbook/assets/image (194).png" alt=""><figcaption></figcaption></figure>



### BGM의 재생·정지 조건

BGM을 업로드했으면 재생·정지 조건을 설정하세요.

1. 「재생 조건 추가」를 누릅니다.
2. 팝업 화면이 열리면 재생을 시작하고 싶은 단계를 선택합니다. 대본 읽기 중 특정 대사부터 재생하고 싶다면 「단계 진행」 부분을 「큐」로 변경하고 큐를 지정합니다.
3. 「변경을 저장」합니다.

<figure><img src="../../.gitbook/assets/image (195).png" alt=""><figcaption></figcaption></figure>



정지 조건을 설정할 때는 재생 조건과 동일하게 「재생 조건 추가」를 누른 뒤, 「결과 액션」 화면에서 「BGM 재생」을 「BGM 정지」로 변경합니다.

<figure><img src="../../.gitbook/assets/image (196).png" alt=""><figcaption></figcaption></figure>



실제 설정 이미지입니다. 

어떤 BGM의 「멈추는 타이밍」으로 설정한 단계가 다른 BGM의 「시작하는 타이밍」으로 설정되어 있으면, BGM이 끊기지 않고 재생됩니다.

<figure><img src="../../.gitbook/assets/image (197).png" alt=""><figcaption></figcaption></figure>



### 어떤 단계부터 계속 같은BGM을 재생하고 싶을 때

엔딩부터 감상 세션까지 같은 BGM을 재생하고 싶다면 재생 조건만 설정하세요. 정지 조건이 없는 BGM은 플레이어가 시나리오에서 퇴장할 때까지 재생됩니다. 

<figure><img src="../../.gitbook/assets/image (198).png" alt=""><figcaption></figcaption></figure>



### BGM의 음량 조절

응용 섹션에 있는 [이 페이지](../../advanced/adjust-audio-file.md)를 참조하세요.

우즈 스튜디오에 업로드하기 전에 iMovie(Mac 기본 제공), Media Player(Windows 기본 제공), Audacity(무료 소프트웨어) 등 외부 도구로 미리 음량을 조절해 두는 것도 추천합니다.
