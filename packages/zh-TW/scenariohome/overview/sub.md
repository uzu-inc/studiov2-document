# 子圖片的設定

封面圖片除了關鍵視覺圖外，還可以設定最多10張的子圖片。

<figure><img src="../../.gitbook/assets/image (147).png" alt="" width="563"><figcaption></figcaption></figure>

請按照以下規則設定子圖片。

* 關鍵視覺圖必須設定在最上面，而不是作為子圖片
* 不要上傳違反公序良俗的內容

{% hint style="info" %}
可以作為子圖片設定的例子

* 關鍵視覺圖的其他版本
* 角色介紹圖片
* 介紹劇情傾向和概要的圖片
* 導入漫畫或導入插圖
* 可以截圖使用的遊戲前卡片
{% endhint %}

{% hint style="danger" %}
不能作為子圖片設定的內容

* 含有劇情劇透的內容
* 引導至外部網站的宣傳
* 與劇情無關的內容
* 其他劇情的宣傳
{% endhint %}
