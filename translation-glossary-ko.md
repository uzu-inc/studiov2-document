# 韓国語翻訳用語集

このドキュメントは、`packages/ja` から `packages/ko` へ翻訳・校正するときの用語統一ルールです。

## 基本方針

* サービス名・機能名は、韓国語として自然に読める範囲で固定訳を使う。
* UI 上のボタン名・画面名は、原文の意味を保ちつつ短くする。
* 「マダミス」は文脈に応じて `머더 미스터리` を基本とし、見出しなど短さが必要な場合のみ既存の固有表現に合わせる。
* `ウズ`、`Discord`、`BGM`、`SE` などの固有名詞・略語は翻訳しない。
* Markdown リンク、画像パス、HTML タグ、GitBook の `{% hint %}` 構文は変更しない。
